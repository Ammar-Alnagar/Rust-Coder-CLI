### Installation

```bash
cargo install rust_tui_coder
```

### Configuration

Create a `config.toml` file:

```toml
[llm]
api_key = "your-api-key-here"
api_base_url = "https://api.openai.com/v1"
model_name = "gpt-4"
```

### Run

```bash
rust_tui_coder
```
